Exercise tracker

Create a New User

POST /api/exercise/new-user

Add exercises

POST /api/exercise/add

GET all users: GET /api/exercise/users
Example: [this_project_url]/api/exercise/users

GET user's exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]
Example: [this_project_url]/api/exercise/log?userId={your_id_here}

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number
